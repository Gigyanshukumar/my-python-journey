# day_9 (first_task) (#Salary Increment Calculator)

This is the first project which i have done today.Basically,i have made a device which give the total salary of an employee which includes the House Rent Allowance and Dearness Allowance.
some features of this device are as follows:
1. Compound Assignment Optimization: Correctly implements the += assignment operator to modify and update the original salary variable dynamically, eliminating unnecessary memory allocations.
2. Precise Allowance Calculations: Uses exact arithmetic formulas to calculate a 12\% House Rent Allowance (HRA) and an 8\% Dearness Allowance (DA) relative to the employee's base pay.
3. Granular Financial Breakdown: Displays intermediate financial metrics (individual HRA and DA amounts) clearly on the console before updating the grand total.

# day_9 (second_task) (#Age Eligibility & Voting Status)

This is thesecond project for today in which i have made a device which takes birth of year of the people as input and calculates their current age as well as gives decision that weather the person is eligible for voting or not.
some features of this device are as follows:
1. Dynamic Age Calculation: Automatically computes a user's exact age by subtracting their input birth year from the static current year benchmark (2026).
2. Relational Threshold Check: Utilizes the >= comparison operator to instantly evaluate if the computed age meets the legal voting requirement of 18 years or older.
3. Direct Boolean Pipeline: Outputs the eligibility status cleanly as a True or False boolean value without relying on complex, multi-line conditional branching (if-else).
4. Neat Terminal Interface: Leverages .center(65) for title alignment alongside strategic spacing elements for a polished CLI display.
