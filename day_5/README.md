# day_5 (first_task) (#the three way rotation)

Today is my fifth day of this python coding journey and here's my first code for today which is a three way rotational device use to rotate the numbers by just putting in the values.
Some features of this device's code are as follows:
1. ​Simultaneous Variable Swapping: feature to rotate three variables in a single line of code (y, z, x = x, y, z), eliminating the need for a temporary third variable.
2. Dynamic User Input: Allows users to input any integer values at runtime for a completely dynamic execution experience.
3. Clean and Centered UI: Uses Python's built-in string methods (.center()) to format and present the challenge title neatly in the terminal.

# day_5 (second_task) (#currency_breakdown)

The second project for today is a currency breakdown device which basically helps people to get to know how many notes of 500 came in there input amount in this device. Also, this device calculates the balance amount after the calculation of all 500s in the entered amount.

Some of the main features of this device's code are as follows:
1. ​Smart Currency Denomination: Automatically calculates the exact number of 500 rupee notes contained within any large amount entered by the user.
2. Accurate Remainder Tracking: Uses efficient math logic to instantly find the leftover balance after the 500 rupee notes are accounted for.
3. Optimized Performance: Built using lightweight and fast Python arithmetic operators (Floor Division // for note count and Modulus % for remaining balance) rather than complex loops.
4. Polished Terminal UI: Features a clean layout with a perfectly center-aligned title using the .center(88) method and proper spacing for better readability.
